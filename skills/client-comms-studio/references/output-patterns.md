# Output Patterns

## Internal brief

Include:
- what the client is saying
- what they need from us
- assumptions being made
- main risk or unknown
- recommended next move

## WhatsApp draft

Style:
- short
- direct
- warm
- commercially clear

Pattern:
1. greet naturally
2. state the conclusion fast
3. give only the needed rationale
4. end cleanly

## Email draft

Style:
- slightly more explicit than WhatsApp
- organize by decision, not by stream of consciousness

Pattern:
1. opening acknowledgment
2. recommendation / answer
3. scope / caveat if needed
4. next step

## Voice note script

Style:
- 30-60 seconds by default
- spoken Urdu/English as needed
- no essay cadence

Pattern:
1. address person by name if given
2. one-line conclusion
3. two or three practical points
4. clean finish, no generic filler
